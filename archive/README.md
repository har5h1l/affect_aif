# Archive

This directory preserves exploratory or superseded surfaces that are not part of the default supported workflow.

Contents:

- `scripts/`: one-off research runners and monitoring utilities
- `configs/`: exploratory or superseded config sets grouped by theme
- `legacy_discrete_beta/`: archived standalone discrete-beta prototype and its historical tests

Archive contents are kept for provenance and temporary reference. They are intentionally excluded from the default docs, lint, type-check, and test surface.
