# Documentation Archive

This directory keeps process-history and superseded planning artifacts that are
useful for provenance but not part of the active paper or operating docs.

- `superpowers/`: completed implementation specs and plans from earlier
  cutover/configuration work. These may contain stale paths or retired
  architecture language and should not be used as current guidance.

Archived material is not part of the default agent read order. Promote durable
current facts back into `docs/active/`, `docs/decisions/`, `docs/results/`,
`docs/roadmap/`, or `docs/reference/` before relying on them for active work.
