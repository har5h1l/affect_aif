# Hypotheses

The current public spine treats affective precision as a relationship-specific
model-fitness signal, not a generic reward-improvement mechanism.

| ID | Role |
|---|---|
| H0 | Policy openness: precision can matter only when the policy space is not saturated. |
| H1 | Model fitness: beta tracks predictability over partner value or reward alone. |
| H2 | Deployment: tracked-only controls separate inference from action confidence. |
| H3 | Locality: partner-local beta gives cleaner relationship-specific signals than a shared tracker. |
| H4 | Social allocation: partner-specific precision reorganizes engagement under choice. |
| H5 | Volatility: abrupt social change tests whether confidence deployment helps or harms adaptation. |
| H6 | Profile-style variation: beta dynamics expose computational individual-difference profiles. |

Paper-scale evidence currently emphasizes predictability-over-value, deployment
over inference, and partner-local-over-shared dissociations. Payoff is treated
as a downstream consequence of calibration interacting with task structure.
