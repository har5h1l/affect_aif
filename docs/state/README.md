# Project State

This directory is the context steering wheel for humans and agents.

- `current/mission.md`: active mission, scope, constraints, and stop conditions
- `current/next_runs.md`: exact experiment queue after restructure verification
- `current/blockers.md`: unresolved blockers and decisions that require human input
- `decisions/`: settled architecture and experiment decisions
- `handoffs/`: dated handoff snapshots replacing conductor logs

Treat these files as current operating state. Current evidence is recorded
under `docs/results/` only after completed supported-architecture runs.
