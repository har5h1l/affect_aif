# Follow-Up Experiment Plan

Last updated: 2026-06-06

## Overview

The manuscript now has two interpreted evidence layers:

1. **Core mechanism**: partner-local affective precision as social
   metacognitive deployment.
2. **Individual-differences extension**: gain and prior define computational
   trust-calibration profiles.

H5, H1, and Exp A-D have reached finality and have been interpreted in the
current paper packet. There is no remaining experiment gate for the current
manuscript draft.

## Completed Evidence

### H1 — Model Fitness Confirmation

**Status**: structurally complete and interpreted.

**Current read**: partner-local precision tracks predictive reliability /
action surprisal more strongly than realized reward after active-encounter
controls. Shared beta preserves the ordering with weaker specificity, while
payoff favors no-affect in this probe. This supports the model-fitness
dissociation, not a reward-improvement claim.

**Contingency rule**:

- If reviewers require a stronger reward/exposure control, use the controlled
  diagnostic ladder before changing the mechanism claim.

Diagnostic ladder:

1. corrected active-encounter confirmation;
2. balanced-exposure graded reliability spine;
3. reward-matched graded reliability spine;
4. strict reward-neutral diagnostic.

### H5 — Abrupt Betrayal Confirmation

**Status**: structurally complete and interpreted.

**Current read**: partner-local affect lowers policy entropy and raises joint
accuracy at 30 seeds; the payoff advantage is small and uncertain. This is an
action-sharpening / temporal-dependency result, not a broad reward-improvement
claim and not the older payoff-with-accuracy-cost smoke story.

### Exp A-D — Phenotype Program

**Status**: structurally complete and interpreted at 20 seeds.

Current manuscript read:

- Exp A: alpha expands beta movement, but payoff is non-monotonic.
- Exp B: prior-by-gain quadrants define computational trust-calibration
  profiles.
- Exp C: forgiveness separates reengagement from restored model-fitness
  confidence.
- Exp D: mixed volatility supports a boundary condition; higher gain can raise
  payoff while worsening discrimination and false-positive suppression.
