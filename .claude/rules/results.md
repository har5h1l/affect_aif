# Results and Analysis Rules

- Never silently rewrite the narrative in results_tracking.md
- Always ask the user before changing interpretation of existing results
- Include exact numbers: p-values, effect sizes, confidence intervals
- Results files are append-only — never delete or overwrite
- Distinguish between exploration (small N, suggestive) and confirmation (large N, conclusive)
- Update the hypothesis scorecard with evidence, not speculation
- Flag any result that contradicts a current hypothesis
