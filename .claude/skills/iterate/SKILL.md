Run one full research iteration focused on: `$ARGUMENTS`

Follow the loop:
1. Orient — read docs, results, code to understand state
2. Plan — identify smallest next step
3. Code — make focused changes
4. Test — run pytest, fix failures
5. Experiment — smoke test then full run
6. Analyze — run analysis on results
7. Interpret — compare against hypotheses
8. Document — update docs with findings
9. Commit — checkpoint with descriptive message
